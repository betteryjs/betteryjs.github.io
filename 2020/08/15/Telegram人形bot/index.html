    前言

    项目名字叫 pagermaid，这是一个能够把自己变成 bot 的项目，功能

    很多，比如一键偷图，自动回复等等，命令方式 - 命令（命令发送 - help 获取），比如偷别人的

    贴纸就回复贴纸-sticker就可以偷到手了，更多功能输入- help` 获取帮助，获取命令用法则输入

    -help 命令名称

    比如这样：

    Github项目地址

    PagerMaid

    教程

    准备：一台 VPS Centos7.5

    更新一下系统

    sudo yum -y update
sudo yum -y upgrade

    GIT 项目

    git clone https://github.com/xtaodada/PagerMaid-Modify.git
cd PagerMaid-Modify

    安装依赖

    pip3 install -r requirements.txt

    可能出现报错信息

    psutil/_psutil_common.c:9:20: fatal error: Python.h: No such file or directory
     #include 
                        ^
    compilation terminated.
    error: command 'gcc' failed with exit status 1

    检查

    • 检查gcc是否安装
    sudo -y yum install gcc
    • 再检查python3-devel 或者python2-devel是否安装

    解决方案

    • 如果你使用的是python2
    yum install -y  epel-release && yum -y update
sudo yum install python2-devel
    • 如果你使用的是python3
    yum install -y  epel-release && yum -y update
sudo yum install python3-devel

    注意

    • 直接使用 sudo yum install python-devel 没有作用

    安装以下插件

    • 注意一个包一个包安装 出现错误好查找信息
    zbar ， neofetch ， tesseract 和 ImageMagick 软件包

yum -y install zbar
yum -y install neofetch
yum -y install tesseract
yum -y install ImageMagick
    • 没有这个包需要安装第三方扩展包
    yum install -y  epel-release && yum -y update
    • 安装email_validator
    pip3 install email_validator
    • 安装pagermaid
    pip3 install pagermaid

    修改配置文件

    将配置 config.gen.yml 文件改名为 config.yml

    cp config.gen.yml config.yml

    去 https://my.telegram.org生成 api 填入配置文件，我们只需要复制 api id 和api_hash值

    将这两串数字填入配置文件，按i 进入编辑模式，粘贴好后，按下 esc 输入 shift 加冒号，输入 wq 保存退出

    vim config.yml

    只修改api_key 和api_hash

    启动项目

    python3 -m pagermaid
